## 7-23 求1！+2！+……+n！ (10 分)
求1！+2！+3！+……+n！，n<12,要求用一重循环设计程序。

**输入格式:**

输入一个小于12的正整数。

**输出格式:**

在一行中以“m=运算结果”顺序输出，其中运算结果是一个正整数，没有任何列宽控制。

**输入样例:**

5
**输出样例:**

m=153

源码如下：

```
#include<stdio.h>
int main()
{
	int i,j,s,m,n;
	scanf("%d",&n);
	m=0;
	for(i=1;i<=n;i++) {
	  	s=1;
	 	for(j=1;j<=i;j++)
		  s=s*j;
	  	  m=m+s;
		}
	}
      printf("m=%d",m);
      return 0;
}
```
