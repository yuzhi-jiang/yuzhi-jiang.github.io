输入一个英文单词（由六个小写英文字母组成），按照下列过程将该单词加密：先将英文单词中的小写字母转换为对应的大写字母，再将该大写字母的ASCII码对10整除后取其余数，从而得到一个六位整数密码。

**输入格式:**

输入在一行中给出1个由六个小写英文字母组成的英文单词。

**输出格式:**

按照规则转换成一个六位整数密码，并输出转换的过程。

**输入样例:**
```
friday
```
**输出样例:**
```
f->F->0
r->R->2
i->I->3
d->D->8
a->A->5
y->Y->9
```
分析：可以用数组，但没必要，因为这道题在一个while循环里就可以完成

代码：

```
#include<stdio.h>
int main(){
    char a,b;    
    int i=6,c;    
    while(i--){
            scanf("%c",&b);        
            a=b;        
            if(b>='a'&&a<='z'){
                   b=b-32;        
                   }        
            c=b%10;        
            printf("%c->%c->%d\n",a,b,c);    
            }    
	return 0;
}
```
