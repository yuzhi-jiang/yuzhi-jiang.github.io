## 题目
上三角矩阵指主对角线以下的元素都为0的矩阵；主对角线为从矩阵的左上角至右下角的连线。
本题要求编写程序，判断一个给定的方阵是否上三角矩阵。
输入格式：
输入第一行给出一个正整数T，为待测矩阵的个数。接下来给出T个矩阵的信息：每个矩阵信息的第一行给出一个不超过10的正整数n。随后n行，每行给出n个整数，其间以空格分隔。
输出格式：
每个矩阵的判断结果占一行。如果输入的矩阵是上三角矩阵，输出“YES”，否则输出“NO”。
输入样例：

```cpp
2
3
1 2 3
0 4 5
0 0 6
2
1 0
-8 2
```

输出样例：

```cpp
YES
NO
```




## 代码：


```cpp
#include<iostream>
using namespace std;
int main(void){
    int t;//需要判断的矩阵个数
    cin>>t;
    int tx=t;//复制t的数据，留待打印时使用
    int sum[t]={0};
    while(t){//当t不为0，输入,判断，t--;
        int n;//矩阵的边长
        cin>>n;
        int count=0;//记录对角线下方的不为0的个数，若最后仍为0，即为 矩阵是上三角矩阵
        int temp[10][10];//矩阵边长不超过10；下标，0-9；
        for(int i=0;i<n;i++){//两重for循环，向二维数组中输入数字；并且判断
            for(int j=0;j<n;j++){
                cin>>temp[i][j];
                if(i>j){//第i行的前（i-1）个数，如第3行的前两个数是需要判断的数
					if(temp[i][j]!=0) count++;//若不为0，,count++,得出矩阵不是上三角矩阵
                } 
                
            }
        }
        //判断最后结果，存在sum数组中，数组【i】的值若为1，矩阵是上三角矩阵
        if(count==0){
            sum[t-1]=1;
        }
        else{
        	sum[t-1]=0;
		}
        t--;//下一个矩阵
    }

    for(int i=tx-1;i>=0;i--){//遍历结果并输出每个矩阵的结论
        if(sum[i]==1){
            cout<<"YES\n";
        }
        else if(sum[i]==0){
            cout<<"NO\n";
        }
    }



    return 0;
}
```
