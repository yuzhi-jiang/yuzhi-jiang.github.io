东软小组成员：素人
本题目要求计算下列分段函数f(x)的值：输入格式:输入在一行中给出实数x。输出格式:在一行中按“f(x) = result”的格式输出，其中x与result都保留一位小数。输入样例1:10
输出样例1:f(10.0) = 0.1
输入样例2:0
输出样例2:f(``0.0) = 0.0


```c
#include<stdio.h>
int main()
{
    float x;
    scanf("%f",&x);
    float y;
    if(x==0)
    {
        y=0;
    }
    else
{
y=1/x;
}
printf("f(%.1f) = %.1f",x,y);
return 0;
}
```
