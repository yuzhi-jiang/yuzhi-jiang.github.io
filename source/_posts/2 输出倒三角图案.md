## 2 输出倒三角图案

本题要求编写程序，输出指定的由“*”组成的倒三角图案。

**输入格式:**

本题目没有输入。

**输出格式:**

按照下列格式输出由“*”组成的倒三角图案。
```c
* * * *
 * * *
  * *
   *

```

分析：输出入门题，可以用for或while循环，~~或者偷懒~~  。

**不偷懒代码：**
```c
#include<stdio.h>
int main()
{
 int a = 4, b = 0, c = 3; 
 for (a = 4; a > 0; a--)//四次循环 
 {
  for (int d = b; d > 0; d--)//前面空格循环 
  {
   printf (" ");
  }
  for(int e = c; e > 0; e--){//*与空格循环 
   printf ("* ");
  }
  printf ("*\n");//*加换行 
  b++;//修改参数 
  c--;//修改参数 
 }
 return 0;
}
```

**~~偷懒代码：~~** 

```c
#include<stdio.h>
int main()
{
 printf ("* * * *\n");
 printf (" * * *\n");
 printf ("  * *\n");
 printf ("   *");
 return 0; 
}
```
