东软学习小组：烈焰
一个整数“犯二的程度”定义为该数字中包含2的个数与其位数的比值。如果这个数是负数，则程度增加0.5倍；如果还是个偶数，则再增加1倍。例如数字-13142223336是个11位数，其中有3个2，并且是负数，也是偶数，则它的犯二程度计算为：3/11×1.5×2×100%，约为81.82%。本题就请你计算一个给定整数到底有多二。

输入格式：
输入第一行给出一个不超过50位的整数N。

输出格式：
在一行中输出N犯二的程度，保留小数点后两位。

输入样例：
-13142223336
输出样例：
81.82%

```csharp
#include <stdio.h>
#include <string.h>
int main() 
{
    char s[55];
    int i, num, two = 0;
    double a = 1.0;
    scanf("%s", s);
    for( i = 0; i < strlen(s); i++ ) {  // 几个2？
    if( s[i] == '2' ) {
    two++;
    }
    }
    num = strlen(s);
    if( s[0] == '-' ) {  // 负数，*1.5，位数-1
    a *= 1.5;
    num -= 1;
    }
    if( (s[strlen(s) - 1] - '0') % 2 == 0 ) {  // 偶数， *2
    a *= 2;
    }
    a *= (two * 1.0 / num * 100);  // 整数除法，注意 * 1.0
    printf("%.2f%%\n", a);

    return 0;
}

```
