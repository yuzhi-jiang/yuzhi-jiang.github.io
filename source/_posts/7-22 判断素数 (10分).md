本题的目标很简单，就是判断一个给定的正整数是否素数。

输入格式：
输入在第一行给出一个正整数N（≤ 10），随后N行，每行给出一个小于2
​31
​​ 的需要判断的正整数。

输出格式：
对每个需要判断的正整数，如果它是素数，则在一行中输出Yes，否则输出No。

输入样例：
2
11
111

      
    
输出样例：
Yes
No


```
#include<stdio.h>
#include<math.h>
int main()
{
    int n, i, flag;
    long long x;
    scanf("%d", &n);
    while(n--)
    {
        scanf("%lld", &x);
        flag = 1;
        for(i = 2; i <= sqrt(x); i++)
        {
            if(x % i == 0)
            {
                flag = 0;
                break;
            }
        }
        if(x == 1 || flag == 0)
            printf("No\n");
        else
            printf("Yes\n");
    }
    return 0;
}

```
