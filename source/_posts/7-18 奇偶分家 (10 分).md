东软学习小组：烈焰
给定N个正整数，请统计奇数和偶数各有多少个？

输入格式：
输入第一行给出一个正整N（≤1000）；第2行给出N个非负整数，以空格分隔。

输出格式：
在一行中先后输出奇数的个数、偶数的个数。中间以1个空格分隔。

输入样例：
9
88 74 101 26 15 0 34 22 77
输出样例：
3 6

```csharp
#include<stdio.h>
int main()//算奇数就可以了，偶数一减就是了
{
    int N, odd=0, i, t;
    scanf("%d",&N);
    for(i = 0; i < N; i++)
    {
        scanf("%d",&t);
        if(t&1)odd++;
    }
    printf("%d %d", odd, N-odd);
    return 0;
}
```
