7-12 计算阶乘和 (10 分)

Author: yuzhi Date: November 3, 2019  13:12:36

对于给定的正整数N，需要你计算 S=1!+2!+3!+…+N!。
输入格式：
输入在一行中给出一个不超过10的正整数N。
输出格式：
在一行中输出S的值。

输入样例：
3

输出样例：
9

code:

#include<iostream>
using namespace std;
int sum(int i){//自身迭代，求 返回i！
    if(i!=1){
        return i*sum(i-1);
    }
    return 1;
}
int main(){
    int n;
    unsigned int s=0;
    cin>>n;
for(int j=1;j<=n;j++){
 s+=(sum(j));
}
cout<<s;
    return  0;
}
